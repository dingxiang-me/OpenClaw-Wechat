# OpenClaw WeCom CLI

```bash
npx -y @dingxiang-me/openclaw-wecom-cli install
```

This thin installer wraps the onboarding APIs from `@dingxiang-me/openclaw-wechat` and can:

- install the plugin through `openclaw plugins install`
- write starter WeCom config into `~/.openclaw/openclaw.json`
- optionally run a post-install `doctor`
- forward `quickstart`, `migrate`, and `doctor` commands from the plugin package
